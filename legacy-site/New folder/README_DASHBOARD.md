# Dashboard System - دليل الاستخدام

## نظرة عامة
نظام لوحة تحكم لإنشاء روابط محدودة للعملاء مع خيارات التحكم في العرض.

## الملفات الرئيسية

### 1. `dashboard.html`
لوحة التحكم الرئيسية لإنشاء وإدارة الروابط.
- **الوصول**: افتح `dashboard.html` في المتصفح
- **البيانات الافتراضية**: 
  - اسم المستخدم: `admin`
  - كلمة المرور: `admin123`

### 2. `viewer.php`
صفحة العرض للعملاء مع الحماية بكلمة مرور.
- **الوصول**: `viewer.php?link=LINK_ID`
- يتطلب تسجيل دخول قبل عرض المحتوى

### 3. `auth.php`
معالج المصادقة للوحة التحكم والروابط.

### 4. `link_manager.php`
إدارة الروابط (إنشاء، عرض، حذف).

### 5. `data.xml`
ملف XML لتخزين المستخدمين والروابط.

## الميزات

### 1. إنشاء روابط محدودة
- إظهار/إخفاء الأسعار
- إظهار/إخفاء الكميات
- فلاتر محددة مسبقاً (نوع المادة، الفئة العمرية، الشركة المصنعة)

### 2. الحماية
- حماية لوحة التحكم بكلمة مرور
- حماية كل رابط بكلمة مرور منفصلة

### 3. التخزين
- استخدام ملف XML بدلاً من قاعدة البيانات

## خطوات الاستخدام

### 1. تسجيل الدخول للوحة التحكم
1. افتح `dashboard.html`
2. أدخل اسم المستخدم وكلمة المرور
3. اضغط "تسجيل الدخول"

### 2. إنشاء رابط جديد
1. في لوحة التحكم، املأ النموذج:
   - اسم الرابط
   - حدد خيارات العرض (السعر، الكمية)
   - حدد الفلاتر المطلوبة
   - أدخل اسم المستخدم وكلمة المرور للرابط
2. اضغط "إنشاء الرابط"
3. سيتم إنشاء رابط فريد مثل: `viewer.php?link=link_xxxxx`

### 3. استخدام الرابط
1. شارك الرابط مع العميل
2. عند فتح الرابط، سيُطلب من العميل تسجيل الدخول
3. بعد تسجيل الدخول، سيتم عرض المواد حسب الإعدادات المحددة

## هيكل ملف XML

```xml
<data>
    <users>
        <user type="admin">
            <username>admin</username>
            <password>hashed_password</password>
        </user>
    </users>
    <links>
        <link id="link_xxxxx">
            <name>اسم الرابط</name>
            <showPrice>true/false</showPrice>
            <showQuantity>true/false</showQuantity>
            <username>username</username>
            <password>hashed_password</password>
            <filters>
                <keyword>...</keyword>
                <materialTypes>...</materialTypes>
                ...
            </filters>
        </link>
    </links>
</data>
```

## الأمان

- كلمات المرور مشفرة باستخدام `password_hash()`
- جلسات PHP للمصادقة
- التحقق من الصلاحيات قبل الوصول للوحة التحكم
- كل رابط محمي بكلمة مرور منفصلة

- احتفظ بنسخة احتياطية من ملف `data.xml` بانتظام
